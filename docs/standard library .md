| 13 | `sys.exit`                | Terminates the VM with an exit code.                       | Exit Code      |            |            | -                          |                                                            |
| 14 | `println`                 | Prints a string to the console with a newline.             | String Address |            |            | -                          | String must be null-terminated.                            |
